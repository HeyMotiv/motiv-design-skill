Three sections, one page
The Why, Habits, and Products map 1:1 to the follower's Essentials section. Same order, same count limits (1 quote, 3 habits, 8 products). No navigation between sub-pages — creators scroll once to see everything.
Inline edit vs sheet
The Why is inline (textarea), habits open a small inline editor (not shown — tap a tile). Products open a full bottom sheet because the form is longer (name, brand, category, tagline, URL, discount). This matches iOS patterns for "short edit vs detailed form."
Save model
Changes are staged until the creator taps Save. The eye icon previews the collab card as followers would see it. Once saved, the home footer link flips from orange dot to green, and the collab's Essentials section updates instantly — no re-publish needed.
Optional by design
Any section can be empty. Empty Why → no quote. Empty habits → no habit tiles. Empty products → no product list. If all three are empty, the follower's Essentials section is hidden entirely, which is why this page can skip being a blocker.
