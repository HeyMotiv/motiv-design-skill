Optional, but monetizable
Essentials don't block going live, but they're a real revenue lever — products with affiliate links drive referral income. The blocker card copy leads with that upside: "Optional — but a great way to boost your referral income." Same visual weight as W9 / Instagram blockers so it doesn't get ignored.
Why a persistent footer link?
Creators need to come back and update essentials over time — add new products, rotate tags, edit copy. The footer link sits alongside Creator Community using the same serif-title row pattern, with the subtitle swapping from "Add habits and products" (empty) to "3 habits · 3 products" (filled).
Stacking with other blockers
The home screen already has a "0 or 1 blocker, prominent" rule. Essentials slots into that queue behind go-live blockers (W9, Instagram, dates). If a critical blocker is present, essentials drops to only the footer link. A small dot on the Profile tab signals there's setup to finish without hijacking the Collabs view.
